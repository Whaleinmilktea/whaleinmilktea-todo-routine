# My Todo App

Todo 관리를 위한 웹 애플리케이션입니다.

## Features

- 편리한 UI로 Todo를 쉽게 생성하고 삭제할 수 있습니다.
- 매일 해야하는 루틴과 데일리로 추가되는 TODO를 관리할 수 있으며, 실천에
참고할 날씨를 확인할 수 있습니다.
- 루틴과 TO DO로 이루고자 하는 목표를 작성할 수 있습니다.
- create-react-app으로 간편한 번들링과 배포가 가능합니다.

## Stack

- Javascript ES6
- React.js 18.2.0
- Webpack
